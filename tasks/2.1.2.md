# Задание 2.1.2

Сформируйте состав описательных и ключевых атрибутов следующих сущностей:

- Учебные дисциплины
- Образовательные уровни
- Формы обучения
- Специальности

для автоматизированной системы планирования учебного процесса в университете.

## Основные понятия

- **Первичный ключ** - идентифицирует один конкретный экземпляр (уникально).
- **Вторичный ключ** - идентифицирует группу экземпляров.
- **Описательные атрибуты** - это всё, что хранит информацию об объекте, но не
  нужно для его уникального распознавания.

## Решение

### Сущность "Учебные дисциплины"

| Тип атрибута | Атрибут             | Описание                    |
| ------------ | ------------------- | --------------------------- |
| Ключевой     | ID_Дисциплины       | Идентификатор дисциплины    |
| Описательный | Название дисциплины | Полное название дисциплины  |
| Описательный | Краткое название    | Сокращённое название        |
| Описательный | Описание            | Содержание дисциплины       |
| Описательный | Количество часов    | Общая трудоёмкость          |
| Описательный | Тип дисциплины      | Обязательная / элективная   |
| Описательный | Преподаватель       | Ответственный преподаватель |

### Сущность "Образовательные уровни"

| Тип атрибута | Атрибут                          | Описание                               |
| ------------ | -------------------------------- | -------------------------------------- |
| Ключевой     | ID_Образовательного_уровня       | Идентификатор уровня                   |
| Описательный | Название образовательного уровня | Бакалавриат, магистратура, специалитет |
| Описательный | Описание                         | Характеристика уровня                  |
| Описательный | Срок обучения                    | Нормативный срок (в годах)             |

### Сущность "Формы обучения"

| Тип атрибута | Атрибут            | Описание                     |
| ------------ | ------------------ | ---------------------------- |
| Ключевой     | ID_Формы_обучения  | Идентификатор формы обучения |
| Описательный | Тип формы обучения | Очная, заочная, очно-заочная |
| Описательный | Описание           | Особенности формы обучения   |

### Сущность "Специальности"

| Тип атрибута | Атрибут                | Описание                    |
| ------------ | ---------------------- | --------------------------- |
| Ключевой     | ID_Специальности       | Идентификатор специальности |
| Описательный | Название специальности | Полное название             |
| Описательный | Направление подготовки | Область обучения            |
| Описательный | Описание               | Содержание специальности    |
| Описательный | Квалификация           | Бакалавр / магистр          |
