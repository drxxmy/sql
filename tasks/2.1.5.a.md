# Задание 2.1.5.a

Сформируйте состав описательных и ключевых атрибутов для сущности **Клиенты**
для автоматизированной системы "Библиотека".

## Решение

### Сущность "Клиенты"

| Тип атрибута | Атрибут                    | Описание                                            |
| ------------ | -------------------------- | --------------------------------------------------- |
| Ключевой     | ID_Клиента                 | Идентификатор клиента библиотеки                    |
| Описательный | ФИО                        | Полное имя клиента                                  |
| Описательный | Дата рождения              | Используется для учета возраста и льгот             |
| Описательный | Адрес                      | Место проживания клиента                            |
| Описательный | Телефон                    | Контактный номер телефона                           |
| Описательный | Email                      | Электронная почта                                   |
| Описательный | Номер читательского билета | Внутренний номер документа пользователя             |
| Описательный | Дата регистрации           | Когда клиент был зарегистрирован в библиотеке       |
| Описательный | Статус клиента             | Активный / заблокирован / временно не обслуживается |
| Описательный | Тип клиента                | Студент / преподаватель / внешний пользователь      |
